NativeYoutube is a lightweight menu bar YouTube client for macOS. It gives you quick playback, playlist support, and a clean native experience without opening YouTube in a browser tab.

---

## Requirements

- macOS 14.0 or later
- A YouTube Data API key

## Quick Setup

1. Open NativeYoutube (menu bar app).
2. Click the gear icon and paste your API key.
3. Start searching and playing videos.

## Add a Custom Playlist

1. Open any playlist URL, for example:
   `https://www.youtube.com/watch?v=dQw4w9WgXcQ&list=PL634F2B56B8C346A2`
2. Copy the value after `list=` (for this example: `PL634F2B56B8C346A2`).
3. Paste it in:
   `NativeYoutube > Settings (Command + ,) > Custom Playlist ID`
